## 0. 一句话定位

构建一个**针对普通话朗读 / 说话的发音测试与纠正系统**：用户对着麦克风读出给定文本（或自由说话），系统从**准确度、流利度、声调、韵律**等维度自动打分，定位读错的字与声调，并给出改进建议及**标准发音示范（TTS）**。打分维度对标国家《普通话水平测试 (PSC)》，算法有效性用 PCC（与人工评分的相关性）验证。

---

## 1. 项目背景与价值

- **真实需求**：普通话水平测试 (PSC) 复习、对外汉语教学、儿童正音、主持/朗诵练习。机辅普通话测试已是国家正式测试的主要方式（单音节、多音节、朗读等项由计算机自动评分），说明"机器自动评估普通话发音"是已规模化的真实场景，而非空想题目。
- **课程契合**：深度覆盖分帧、加窗、预加重、STFT、MFCC/FBank、短时能量、过零率、基频(F0)、滤波、DTW 等课程核心知识点，并集成 ASR、TTS。
- **加分点**：
  1. **声调（F0 轮廓）评分**——英文发音评测体系里没有的维度，普通话独占，最显课程功底；
  2. **多维打分对标 PSC 官方标准**——应用价值有官方依据；
  3. **PCC 客观验证**——用学术界公认指标证明打分不是"拍脑袋"。

## 3. 总体功能

**主线功能（朗读模式，核心）：**
1. 选择 / 录入朗读文本（系统自动生成拼音 + 声调标注）。
2. 麦克风录音（或上传音频）。
3. 自动测试，输出：总分 + 分维度分（准确度 / 流利度 / 声调 / 韵律 / 完整度）。
4. **逐字诊断**：标出读错的字、错误的声调，定位不流畅的片段。
5. **可视化**：用户 F0 曲线 vs 标准 F0 曲线、波形、频谱、停顿分布。
6. **标准发音示范**：对错误字用 TTS 合成标准读法，供对比跟读。
7. 生成测试报告（可导出）。

**扩展功能（自由说话模式，对应 PSC 命题说话）：**
- 无参考文本，只评流利度 + 韵律 + 语气 + 停顿，给"自然流畅度"分。

**可选加分扩展（时间允许）：**
- 说话人识别（声纹）：多用户成绩库存档；
- 情感/语气分析：朗读情感丰富度。

## 4. 总体架构与处理流程

### 4.1 数据流（朗读模式）

```
参考文本 ───► [pypinyin] ───► 拼音+声调序列（带音调）
                                        │
麦克风/音频 ───► 预处理 ───► 特征提取 ──────┤
              (重采样/预加重/VAD)  (MFCC/F0/能量)
                                        │
                            强制对齐(拿到每个字/音素的时间边界)
                                        │
              ┌──────────────┬───────────┴──────┬───────────────┐
              ▼              ▼                   ▼              ▼
          准确度打分      声调打分            流利度打分      韵律打分
          (DTW/GOP)    (F0轮廓分类)        (语速/停顿)     (F0周期/语调)
              └──────────────┴───────────┬──────┴───────────────┘
                                        ▼
                              综合打分(映射PSC维度)
                                        │
                       ┌─────────────────┼─────────────────┐
                       ▼                ▼                ▼
                  逐字诊断报告      可视化(F0/波形)    TTS标准示范
```

### 4.2 处理流程五步

1. **采集与预处理**：录音 → 重采样 16kHz 单声道 → 预加重 → VAD 去首尾静音 → 分帧加窗。
2. **识别与对齐**：ASR 拿识别文本（校验/完整度）；强制对齐拿到每个音节/音素的起止时间。
3. **特征提取**：逐帧 MFCC/FBank、F0(基频)、短时能量、过零率。
4. **多维打分**：准确度、声调、流利度、韵律、完整度，加权得总分。
5. **反馈与报告**：逐字诊断 + 可视化 + TTS 示范 + 报告导出。
